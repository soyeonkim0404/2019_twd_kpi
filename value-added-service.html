안내 사항

  • 모바일을 기준으로 하는 이유는 css 값이 현재 상용서버 내부 소스에 포함되어 어드민 등록시 추가적인 css 삽입을 하지 않기 때문이다.
  • PC 버전의 경우 아직 Css작업 진행 되지 않아서, 어드민 등록시 필요한 css를 포함하여 등록한다.
  • 어드민 시작 / 끝 이라는 주석 내부 소스가 실제 어드민에 등록 되는 소스이며, To-be 사이트로 전환 이후에도 As-is 사이트의 소스들이 일부 사용 되고 있다.
  • To-be 사이트의 실제 적용 페이지를 전달 받지 못하여, 실제 적용 된 모습을 알 수가 없으므로 향후 전달 받아서 CSS 정리가 필요하다.
  • 현재 소스에 가장 큰 특징은 테이블 col(.table-term) / row (.table-term-row) 형태에 따라 클래스가 나뉜다.
  • 제공되는 소스는 유형별 마크업이며, 새창열기를 통해 제공되는 실제 파일을 참조.

* PC 버전과 모바일 버전의 마크업은 거의 유사하며, 약간의 차이가 있다.
향후 고도화 작업을 통해 공통 소스로 맞출 필요가 있다.
공통 소스로 맞추면 굳이 pc / mobile 두벌씩 관리 할 필요가 없다.
